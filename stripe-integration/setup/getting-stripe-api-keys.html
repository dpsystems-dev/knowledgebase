How to Get Your Stripe API Keys

The secret key allows your server to communicate with Stripe's API. This guide shows you how to retrieve it from the Stripe Dashboard.

Prerequisites​

Before you begin, ensure you have:

  • Access to a Stripe account (dashboard.stripe.com)
  • Administrator access to the Stripe account

How to Find the Secret Key (Sandbox Mode)​

What you'll do: Navigate to the API keys section and copy your secret key.

Step 1: Log into Stripe Dashboard​

  1. Go to dashboard.stripe.com and log in to your Stripe account

Step 2: Verify Sandbox Mode​

  1. Confirm you are in Sandbox mode by checking for the blue "Sandbox" label in the top-left corner of the dashboard

Stripe Dashboard showing Sandbox mode and API keys section

Step 3: Locate the API Keys Section​

  1. On the dashboard, look for the "API keys" section on the right side of the page

Step 4: Copy the Secret Key​

  1. Find the "Secret key" row. The key is hidden by default and shows as dots

  2. Hover over the Secret key row and click the "Click to copy" button that appears

  3. The key is now copied to your clipboard. It should start with sk_test_

Keep Your Key Secure

The secret key allows full access to process payments on your account. Never share it publicly, commit it to version control, or expose it in client-side code.

Understanding Key Types​

Key PrefixModePurpose
sk_test_SandboxTest payments (no real money)
sk_live_LiveReal payments
tip

Always test in Sandbox mode first before switching to Live mode.

Live Mode Key Differences​

When you're ready for real payments, you'll switch to Live mode. Key differences:

AspectSandbox ModeLive Mode
Key prefixsk_test_sk_live_
Secret key revealCan reveal multiple timesCan only reveal ONCE
PaymentsTest/simulatedReal money
Live Mode Warning

In Live mode, the secret key can only be revealed once. Copy it immediately and store it securely. If you lose it, you'll need to create a new key.

Next Steps​

After copying your secret key:

  1. Create a Webhook Endpoint - Set up notifications
  2. Configure the Portal - Enter the key in Admin Dashboard
