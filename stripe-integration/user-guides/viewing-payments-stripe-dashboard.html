How to View Payments in Stripe Dashboard

Administrators can view all payment transactions in the Stripe Dashboard for reconciliation and reporting purposes.

How to Access the Stripe Dashboard​

What you'll do: Log in and navigate to the transactions list.

  1. Log in to the Stripe Dashboard at dashboard.stripe.com
  2. Navigate to Transactions in the left menu

Stripe Transactions page showing all payments

How to View Payment Details​

What you'll do: View detailed information about a specific payment.

Click on any payment to view its details:

Payment details page showing transaction information

The payment details page shows:

  • Payment amount and status
  • Customer information
  • Checkout summary with line items
  • Payment breakdown (fees, net amount)
  • Payment method details
  • Timeline of events

How to Use Payment Metadata​

What you'll learn: How to match payments back to the ABM Service Portal.

Each payment includes metadata that links it back to the ABM Service Portal:

Metadata section showing portal-specific information

Metadata Field Reference​

FieldDescription
createdByEmailEmail of the user who initiated the payment
customerCodeThe customer code in the portal (e.g., "000001")
customerIdThe unique customer ID in the system
customerNameThe customer's name (e.g., "Limited Energy")
paymentIdThe internal payment ID in the portal
referenceThe payment reference entered by the user
sourceRoleWho initiated the payment: "abm" (internal user) or "external" (customer)

How to Use Metadata for Reconciliation​

This metadata makes it easy to:

  • Track which portal user processed the payment
  • Match payments to customer accounts using customerCode
  • Reconcile payments with invoices using the reference field
  • Distinguish between internally-processed and customer-initiated payments using sourceRole
